# Spendenmodell & Anonymität

## Prinzip

Spenden sind **freiwillig und anonym**. Kein Account, keine Identität nötig.

## Token-System

```
1. Spende leisten (Crypto oder PayPal)
2. System generiert anonymen Zugangstoken
3. Token aktiviert KI-Server + Feature Requests
4. Keine Verbindung zwischen Token und Person
```

## Zahlungswege

| Methode | Anonymität |
|---------|-----------|
| Monero (XMR) | ⭐⭐⭐ Maximal |
| Bitcoin (BTC) | ⭐⭐ Hoch |
| PayPal | ⭐ Gering |

## Preise

| Spende | Zugang | Dauer |
|--------|--------|-------|
| €3 | KI-Server + Feature Requests | 1 Monat |
| €25 | KI-Server + Feature Requests | 12 Monate |
| €10–50 | Feature Wunsch Boost | Einmalig |

## Transparenz

Monatlich veröffentlicht: Gesamteinnahmen, Serverkosten, Anzahl aktiver Token.
Nie veröffentlicht: Wer gespendet hat, welcher Token zu wem gehört.
