# Wie Anonymität funktioniert

*Für alle die es genauer wissen wollen — ohne technisches Vorwissen*

---

## Das Grundprinzip

HealthLedger kennt dich nicht. Das ist kein Versehen — es ist Absicht.

Du brauchst keinen Account, keine E-Mail-Adresse, keinen Namen.
Deine Identität ist ein zufälliger Schlüssel der auf deinem Gerät erzeugt wird.
Niemand außer dir kennt die Verbindung zwischen diesem Schlüssel und deiner Person.

---

## Wie das Token-System anonym funktioniert

Stell dir vor du kaufst eine Prepaid-Karte — aber so dass
der Verkäufer nicht weiß wer sie kauft, und der Betreiber
nicht weiß wer sie nutzt.

So funktioniert es:

**Schritt 1 — Du zahlst anonym**
Du überweist per Monero (eine Kryptowährung die keine Spuren hinterlässt)
oder per Bitcoin. Oder per PayPal wenn dir das wichtig ist.

**Schritt 2 — Du bekommst einen Einmal-Code**
Nach der Zahlung bekommst du einen zufälligen Code.
Dieser Code hat keine Verbindung zu deiner Zahlung — er ist
wie eine Banknote: wer sie hat, kann sie nutzen, egal wer sie hatte.

**Schritt 3 — Du löst den Code ein**
Mit dem Code aktivierst du dein Token-Guthaben.
Das System sieht nur: „Jemand hat gültigen Code eingelöst."
Es weiß nicht wer gezahlt hat.

**Schritt 4 — Du nutzt die KI-Funktionen**
Dein Guthaben wird verbraucht wenn du KI-Features nutzt.
Das System sieht nur deinen anonymen Schlüssel und das Guthaben.

---

## Das Zwei-Schlüssel-System (für maximale Anonymität)

Wer es noch sicherer will, kann zwei völlig getrennte Schlüssel nutzen:

- **Schlüssel A** nur für die Zahlung
- **Schlüssel B** nur für die Nutzung

Beide kennen sich nicht. Selbst wenn jemand die Zahlung
zurückverfolgen könnte — er kann nie herausfinden wer Schlüssel B ist
und was damit genutzt wurde.

---

## Warum das wichtig ist

Gesundheitsdaten sind besonders sensitiv. Wer weiß dass du
bestimmte Gesundheitsfragen stellst, weiß viel über dich.

Unser Versprechen: Nicht mal wir können das herausfinden.
Und weil der Code öffentlich ist, kann jeder prüfen ob das stimmt.
